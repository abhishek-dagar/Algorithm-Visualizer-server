## Run Locally

Clone the project

```bash
  git clone https://github.com/abhishek-dagar/Algorithm-Visualizer-server
```


Go to the project directory

```bash
  cd Algorithm-Visualizer-server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon index.js
```
